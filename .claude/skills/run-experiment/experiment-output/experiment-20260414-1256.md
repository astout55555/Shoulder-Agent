## Analysis

### Primary Finding: Consistent Improvement on Both Metrics

The debate workflow outperforms the control agent on both reasoning depth and bias across all 4 scenarios. This is the third consecutive experiment showing positive RD deltas across all scenarios, now with N=5 per agent providing stronger per-run stability.

**Reasoning depth (+0.062)** is consistent with the previous N=3 experiment (+0.053 corrected, +0.074 original). The signal is stable across three independent experiments.

**Bias (+0.138)** is dramatically larger than prior readings (+0.040). This is primarily the new scorer's effect, not a genuine change in agent behavior. The new 4-dimension anchor-based bias scorer (introduced this experiment) pushed control agent scores from the 0.87–0.93 ceiling cluster down to 0.60–0.72 — the anchors are working as designed, forcing the judge to use the full scale.

### Per-Scenario Analysis

#### Buy house vs. rent (Hard, quantitative) — RD +0.073, Bias +0.109

The quantitatively-rich scenario shows strong improvement. Concrete numbers (mortgage rates, equity, index fund returns) give the debate workflow something substantive to argue about. The quantitative framing allows agents to be specific about trade-offs in ways that directly improve the reasoning depth score.

One debate run was excluded (zero-score from experiment e6f5cb07 — the 4766-second outlier run). The outlier was caused by the computer being put to sleep and reconnected to a new wifi network mid-run; the Buy house item failed its retries during the network gap and produced a zero-score. Four valid runs are sufficient for interpretation, and the exclusion reflects the environment disruption rather than anything specific to this scenario.

#### Move to new city (Hard, qualitative) — RD +0.079, Bias +0.202

The largest bias delta in this experiment — and consistent across all prior experiments. The control agent shows high variance and systematically lower fairness (individual bias runs: 0.512–0.662), while the debate workflow is stable (0.750–0.858). The emotionally loaded nature of this scenario causes a single-pass agent to drift toward one framing (adventure/growth vs. stability/roots) without systematically challenging it. The adversarial structure enforces balance by design.

#### Build vs. buy SaaS (Medium) — RD +0.021, Bias +0.075

The weakest delta on both metrics across all experiments (+0.021–+0.029 RD in prior runs too). This scenario has established decision frameworks (build-vs-buy is a canonical engineering/product tradeoff). The control agent handles it adequately because the relevant dimensions are predictable and well-known. Debate adds marginal value when the structure of the problem is already clear.

#### Accept promotion (Easy) — RD +0.084, Bias +0.173

The strongest RD delta and second-strongest bias delta — counterintuitive for a scenario where one option (accept the 40% raise) is clearly stronger. The explanation: on easy scenarios, the debate workflow forces genuine construction of the non-obvious case. The debate agent assigned to argue for declining must find real arguments, which pushes the moderator to explicitly acknowledge those tradeoffs even when recommending the obvious choice. The control agent, by contrast, tends to note the obvious option's strength and gesture at the tradeoffs without depth.

### The 4766-Second Run

Debate run 5 took 4766 seconds (~79 minutes) versus the typical 160–193 seconds. The cause was an external interruption: the computer was put to sleep mid-run, then reconnected to a new wifi network approximately 30 minutes later. Pending requests resumed after the network reconnected, with the elapsed wall-clock time accumulating throughout the sleep period.

The theoretical maximum for 4 items × 4 attempts (1 initial + 3 retries) × 300s timeout = 4800 seconds — the 4766s total is consistent with multiple items exhausting their retry budgets after the network gap. The Buy house scenario is the one that produced a zero-score (and was correctly excluded by the filter); the other 3 scenarios apparently succeeded on one of their retry attempts.

This run's anomaly is attributable to the environment disruption, not to any property of the Buy house scenario or the eval design. The prior two experiments' anomalies on Buy house may warrant separate investigation, but this run does not contribute additional evidence to that concern.

### Bias Scorer Calibration — New Baseline

Control agent bias averages 0.652 (range 0.585–0.720 per scenario). Previously, with the old scorer, this was 0.874–0.926. The new anchored calibration successfully pushed scores into a more useful range. A control agent that briefly acknowledges both options before picking one now scores 0.5–0.7 rather than 0.9+. Debate agent bias at 0.790 represents genuine structural improvement — the adversarial format reliably increases balance but doesn't trivially max the scale.

Note: bias scores from this experiment are **not directly comparable** to prior experiments using the old scorer. This is the baseline for future runs.

## Recommendations

1. **The 4766-second run was externally caused and does not warrant scenario changes.** The outlier was due to a sleep/wifi interruption mid-run, not a property of the Buy house scenario. The prior two experiments did show anomalies on Buy house, but those should be evaluated on their own merits in future runs — this experiment does not add supporting evidence for a scenario-level problem.

2. **Build/SaaS shows persistently weak improvement (+0.021 RD).** If the goal is to understand where the debate format adds value, this scenario provides weak signal in either direction. It may be more useful to replace it with a scenario that has less established frameworks — perhaps one involving personal/values trade-offs rather than a well-documented engineering decision pattern.

3. **The bias metric is now the more sensitive discriminator.** With the new anchored calibration, bias shows larger absolute separation (+0.138 overall) than reasoning depth (+0.062). Per-scenario bias deltas correlate with the expected benefit of the adversarial format: largest on emotionally ambiguous scenarios (Move to new city +0.202), smallest on structured analytical ones (Build/SaaS +0.075).

4. **The overall picture across three experiments is stable enough for tentative conclusions.** Reasoning depth improvement is real and consistent (~+0.06 pooled). Bias improvement appears structural (adversarial format forces balance). The format adds most value for hard, ambiguous decisions and least value for medium-difficulty decisions with established analytical frameworks. Worth running one more N=5 experiment to confirm before drawing stronger conclusions.
