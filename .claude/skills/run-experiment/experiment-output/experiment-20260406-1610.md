- 2 workflows (debate, control)
- 3 runs per agent (6 experiments total)
- Scorer: high-bar calibration (professional decision coach standard)

## N=3 Results (retroactively recovered)

The initial script output used `compareExperiments` which deduplicated results to N=1 per scenario. The full N=3 data was retroactively recovered by querying the scores store directly via `listScoresByScorerId`, matching scores to experiments by `runId` (experiment ID) and to scenarios by `entityId` (item ID). The script has been permanently fixed to use this approach.

## Analysis

### Reasoning Depth (+0.074 overall) — Strong, Consistent Improvement

With N=3, the debate workflow outperforms the control on **all 6 scenarios** — every per-scenario delta is positive. The N=1 anomaly where "Stay vs. new job" showed -0.001 has resolved to +0.058 at N=3, confirming it was noise.

**Run-to-run variance is moderate but consistent.** Individual run scores for reasoning depth span a range of ~0.05–0.11 within each scenario for both agents (e.g., promotion debate: 0.737–0.847, promotion control: 0.613–0.737). Despite this variance, the debate workflow's 3-run average beats the control's 3-run average on every scenario. This consistency across 18 independent runs is strong evidence that the improvement is real, not a fluke.

**Difficulty gradient at N=3:** Hard (+0.088) and Easy (+0.093) show similar gains; Medium is weakest (+0.042). The hypothesis that debate helps more on hard scenarios is partially supported — Hard scenarios do show strong gains — but the Easy group's equally strong performance complicates the narrative. The Easy group is dominated by the promotion scenario (+0.128), which despite being the "easiest" decision, benefits greatly from the debate format's thorough justification.

**The buy/rent scenario's improvement softened at N=3.** The N=1 reading was +0.153; at N=3 it's +0.097. Still the strongest Hard scenario, and the enriched quantitative context is clearly helping — the debate workflow achieves the highest absolute reasoning depth scores here (0.792) — but the dramatic single-run delta was partly inflated by a weak control run (0.660) that averaged up to 0.695 at N=3.

### Advice Bias (+0.044 overall) — Consistent Improvement

At N=3, the debate workflow matches or exceeds the control on bias for all 6 scenarios. The "Move to new city" scenario remains the standout (+0.153 bias delta), and importantly, the control's variance on this scenario is high (0.750–0.920), suggesting the control agent's fairness is inconsistent for emotionally-charged scenarios. The debate workflow is both better and more stable (0.950–1.000).

### Advice Relevancy (+0.020 overall) — Small Positive

Every scenario shows a non-negative relevancy delta at N=3. The prior concern about the buy/rent relevancy regression (-0.100 at N=1 in run 2) is fully resolved — both agents score 1.000 with the enriched scenario.

### Decisiveness (0.000)

Both agents hit ceiling (1.000) across all 18 runs. This metric provides no discriminating signal.

### Variance Analysis

Notable variance observations:
- **Promotion (Easy):** Debate has the widest reasoning-depth spread (0.737–0.847, range=0.110). Control also has wide spread (0.613–0.737, range=0.124). This scenario may be inherently more variable because the "obvious" answer leads to more diverse justification strategies.
- **Build/Buy (Medium):** Debate has notable spread (0.712–0.787, range=0.075) while control is tighter (0.700–0.750, range=0.050). The debate workflow's adversarial structure may introduce more randomness in how it frames the build-vs-buy tradeoff.
- **Move to new city (Hard):** Control bias has the highest variance of any metric/scenario combination (0.750–0.920, range=0.170). The control agent's fairness is unreliable on this emotionally-valenced scenario.

## Recommendations

1. **The +0.074 reasoning depth improvement is statistically robust at N=3.** The debate workflow consistently outperforms the control across all scenarios, all difficulty levels, and all runs. This validates the Shoulder Agent's core thesis.

2. **The high-bar scorer calibration is effective.** Scores center around 0.65–0.80 with meaningful variance, providing good discriminating power. No further calibration changes needed.

3. **Consider enriching Medium scenarios with more context.** The Medium group (+0.042 RD) lags behind Hard (+0.088) and Easy (+0.093). Adding quantitative detail to the masters/self-learning scenario (specific tuition costs, salary uplift data) could enable more concrete reasoning for the debate workflow to exploit.

4. **The decisiveness metric is uninformative.** Both agents always score 1.000. Consider replacing it with a metric that provides signal, or simply removing it to save scorer computation time.

5. **The script's comparison logic has been fixed** to query scores directly via `listScoresByScorerId` instead of `compareExperiments`, which deduplicated results. Future `/run-experiment` invocations will correctly show all N=3 data.
